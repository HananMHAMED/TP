# Portfolio Projets

Ce dépôt contient plusieurs projets réalisés en développement web, web dynamique, programmation et analyse de données.

## Projets principaux

- Web_dynamique — application web dynamique en PHP avec base de données, architecture MVC et API REST
- Projet_data_mining — projet d’analyse et de traitement de données
- Projet_agence_voyage — site web de réservation de voyages en HTML, CSS et JavaScript
- Jeu_Space_Invaders — jeu en Python inspiré de Space Invaders
-  Projet_chatbot — application Streamlit avec intégration d’API

Chaque projet contient un README détaillé.
